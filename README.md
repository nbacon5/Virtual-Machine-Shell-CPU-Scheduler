# Description
A virtual shell for executing programs and managing their state.

# Features
- Execute or pipe together unix commands or custom C programs in the foreground or background
- Maintains a list of all active processes
- Add, suspend, or kill processes

# Build
- Requires gcc and make
- Run `make` from the project root

# Run:
- Start the virtual shell with ```./anav```
- Instructions for using the shell are displayed in the terminal
